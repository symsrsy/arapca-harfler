# Arapça Harfler

Veritabanı gerektirmeyen, HTML/CSS/JavaScript ile hazırlanmış küçük bir eğitim uygulaması.

## Kullanım

- `index.html` dosyasını tarayıcıda açın.
- **Rastgele Sor** ile yeni bir harf getirin.
- **Cevabı Göster** ile harfin okunuşunu açın.
- Sağ/sol ok tuşlarıyla harfler arasında geçin.
- Boşluk tuşu ile rastgele harf getirin.
- `C` tuşu ile cevabı açıp kapatın.
